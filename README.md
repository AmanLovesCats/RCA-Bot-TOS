# Repuls RCA Bot - Legal Pages

This repository hosts the official Privacy Policy and Terms of Service for the **Repuls RCA Bot**, the official Discord bot developed in association with the Repuls Community Association (RCA) and the developer of [Repuls.io](https://repuls.io).

## Purpose
These pages are intended to be hosted via GitHub Pages to serve as a public, verifiable source for the bot's legal documentation. This is a strict requirement for Discord Bot Verification.
